# 📊 Projeto de Inteligência Artificial - Análise de Comentários do YouTube

## 📌 Descrição
Este projeto tem como objetivo analisar comentários de vídeos do YouTube utilizando técnicas de Inteligência Artificial, com foco em Processamento de Linguagem Natural (NLP).

A proposta busca automatizar a análise de grandes volumes de dados textuais, identificando sentimentos, padrões de comportamento e temas relevantes nos comentários dos usuários.

---

## 🎯 Objetivo
- Coletar dados de vídeos e comentários do YouTube
- Realizar pré-processamento dos dados
- Aplicar técnicas de NLP para análise de sentimentos
- Identificar tópicos e padrões nos comentários
- Extrair informações relevantes como marcas e produtos

---

## 🧠 Metodologia
O projeto foi desenvolvido seguindo um pipeline de dados:

1. Coleta de dados via YouTube Data API  
2. Limpeza e preparação dos dados  
3. Análise exploratória (EDA)  
4. Aplicação de técnicas de NLP  
5. Classificação de sentimentos  
6. Modelagem de tópicos  
7. Extração de marcas e modelos

## 📊 Dataset
Os dados foram coletados utilizando a YouTube Data API, incluindo:

- Informações dos vídeos (visualizações, curtidas, comentários)
- Comentários dos usuários
- Dados processados para NLP
- Tópicos e palavras-chave
- Informações sobre marcas e produtos

⚠️ Observação:  
Devido ao tamanho dos arquivos, alguns datasets foram reduzidos para fins de upload no GitHub.

## ⚙️ Tecnologias Utilizadas
- Python

- Matplotlib / Seaborn

- YouTube Data API

## 📈 Resultados Esperados
- Classificação de comentários em positivo, negativo e neutro  
- Identificação de padrões de comportamento dos usuários

- Análise de percepção do público sobre marcas e produtos
